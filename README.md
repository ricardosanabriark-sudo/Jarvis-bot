# 🤖 Jarvis Bot

Asistente personal con Telegram y Claude AI. Recibe tareas por Telegram, las procesa con IA y envía recordatorios automáticos.

## Variables de entorno necesarias

| Variable | Descripción |
|----------|-------------|
| `TG_TOKEN` | Token del bot de Telegram |
| `TG_CHAT_ID` | Tu Chat ID de Telegram |
| `CLAUDE_API_KEY` | API Key de Anthropic |

## Deploy en Render

1. Conecta este repositorio en render.com
2. Añade las variables de entorno
3. ¡Listo!
